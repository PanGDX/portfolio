# Vim vs Nano

I used to use Nano. Then I found a server where Nano wasn't installed. I had to learn Vim.

Now I can't quit. Literally, `:q!`.

## Keybindings
*   `i`: insert mode
*   `esc`: normal mode
*   `:wq`: save and quit
